# 🚀 Patan Hajee Baba | Personal Portfolio

Welcome to the official repository for my personal portfolio website. This site serves as a digital hub to showcase my skills, projects, and professional journey as a developer.

[**View Live Demo**](https://hajeebaba-portfolio.netlify.app/)

---

## 🛠️ Tech Stack

This portfolio is built using modern web technologies to ensure performance, responsiveness, and a sleek user interface:

* **Frontend:** React.js, Next.js
* **Styling:** CSS3 / Modern Libraries
* **Languages:** Java, Python, SQL
* **Backend / Logic:** Node.js
* **Deployment:** Netlify

---

## ✨ Features

* **Responsive Design:** Optimized for all screen sizes (Desktop, Tablet, Mobile).
* **Dynamic Sections:** Includes an About section, Project showcase, and integrated Resume view.
* **Clean UI:** Minimalist aesthetic with intuitive navigation and smooth animations.
* **Contact Integration:** Easy access to social profiles and professional networks.

## 💻 Installation & Local Development

To get a local copy up and running, follow these simple steps:

### 1. Clone the Repository
bash```

git clone [https://github.com/PatanHajeeBaba/Portfolio.git](https://github.com/PatanHajeeBaba/Portfolio.git) 

### 2. Navigate to the Directory
* cd Portfolio
### 3. Install Dependencies
* Make sure you have Node.js installed. Then run:
* npm install
### 4. Run the Development Server
* npm start or  npm run dev
